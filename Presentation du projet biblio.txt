II -) Developpement du projet biblio

1-) Interface(front-end) : Utilisation d'un template bootstrap 5 appellé Sb Bootstrap Admin 2 disponible sur
https://startbootstrap.com/templates/general, Auquelle nous avons ajouter d'autre interface en utilisant des classes de bootstrap 5 .

2-) Arriere Plan (Backend)

a -> Serveur & Base de Données

Nous avons utliser un serveur Apache et un systeme de gestion de base de données MYSQL grace à un environement de developpement XAMP .

b -> Programmation 

Nous avons utiliser le langage de programmation PHP (Hypertext Preprocessor) Pour la connexion à MYSQL  dans le but d'y recuperer des données que nous renverrons à Apache qui ensuite le retournera au navigateur quand l'utilisateur entrera l'url 
https://biblio.local

c-> Securité

Utilisation des requetes preparer pour proteger les données de la base contre les Injections SQL 

Integration des fontions comme htmlspecialchars() pour proteger l'affichage des données contre les attaques XSS.


II -) Contexte Du Projet

1-> Administrateur

Biblio 2 est une application web permettant a son administrateur d'informatiser la gestion de sa bibliotheque.

2-> Fontionalités

biblio 2 permet à l'admin de gerer sa bibliotheque :

- Voir , Ajouter , Modifier et Supprimer la liste des usagers , les livres et les emprunts

Nous avons utiliser Git et GitHub pour gerer les versions du projet et gerer notre travail en equipe pour cela le chef de projet à envoyer le projet sur le serveur de GitHub et nous a invités à collaborer nous les menbres ainsi que notre formatteur.
